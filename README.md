# CamShield

CamShield is a Webcam Security and Intrusion Detection System built using Python.

## Features

- User Authentication
- Face Recognition
- Intruder Detection
- Email Alerts
- Encrypted Logs
- Webcam Monitoring
- Dashboard Interface

## Technologies

- Python
- PyQt6
- OpenCV
- Face Recognition
- Dlib
- Cryptography
- bcrypt

## Installation

pip install -r requirements.txt

## Run

python -m app.main
